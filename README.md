# 题目：在系统管理下面新增"文章管理"菜单，文章的字段有：
- 标题，字符类型，长度4-50，必填
- 内容，富文本类型，长度0-5000
- 类型，字符类型，可以选择的值有3个： 生活，科技，文化。 必填
- 发布时间，日期类型，格式yyyy-MM-dd HH:mm:ss ，在页面上面使用空间输入
- 文章来源， 字符类型，长度4-20，必填
